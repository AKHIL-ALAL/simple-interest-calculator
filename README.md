# Simple Interest Calculator

This repository contains the required files for the GitHub UI assignment:
- LICENSE
- README.md
- CODE_OF_CONDUCT.md
- CONTRIBUTING.md
- simple-interest.sh

## How to Run the Script
Make the script executable:
```bash
chmod +x simple-interest.sh
````

Run:

```bash
./simple-interest.sh 1000 5 2
```

Output:

```
The simple interest is: 100
```
